Automation API Reference
The Automation API reference documentation is located in the References section. You can access from there:
  • The list of the frameworks exposing objects, each of one pointing to the documentation of all its exposed objects.
  • The object inheritance hierarchy.
  • Indexes, and among them, namely:
    • The index of deprecated objects, methods, and properties.
    • The object index.
    • The object method index.
    • The object property index.

    And other useful indexes about Automation types.
