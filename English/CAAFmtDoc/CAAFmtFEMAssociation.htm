Managing links between the 3D Shapes and the FEM Rep
Technical Article

Abstract

This article's main goal is to help partners understand how to manage links between 3D Shapes and a FEM Rep.

  • Why creating links between FEM Rep and 3D Shapes are needed ?
  • How to create links between FEM Rep and 3D Shapes ?
  • How to remove links between FEM Rep and 3D Shapes ?

Why creating links between FEM Rep and 3D Shapes are needed ?

A FEM Rep must be associated to 3D Shapes in order to use the geometries contained by them as support of FEM features.

  • Geometry can be a volume, a surface, a curve, a line, a vertex...
  • FEM features can be meshes, sections, masses, groups...

How to create links between FEM Rep and 3D Shapes ?

The CATIFmtLinkedRep interface which is implemented on the FEM Rep allows to create or retrieve links between the FEM Rep and the 3D Shapes. The following code snippet shows how to create links from the FEM Rep manager.

...
// Associate the 3D shapes to the FEM Rep
CATFmtListOfPath l3DShapeList;
...
CATIFmtLinkedRep_var spLinkedRep(spRepManager);
hr = spLinkedRep->AddAssociatedReps(l3DShapeList);
...

As shown in this exemple, a list of CATFmtPath must be created before to call the AddAssociatedReps method. The class CATFmtPathFactory allows creation of these path elements from the representation instances of the 3D Shapes. For further information about navigating into the PLM product structure to retrieve the 3D Shape representation instances refer to the article "Navigation on PLM Core data"[1].

...
// Create a CATFmtPath from object
CATFmtPath * p3DShapePath = CATFmtPathFactory::BuildFromObject ( sp3DShapeRepInst );
l3DShapeList.Append(p3DShapePath);
...

How to remove links between FEM Rep and 3D Shapes ?

Similarly, a list of CATFmtPath pointing all the 3D Shape which will be removed must be created first. Then the RemoveAssociatedReps method can be called to remove the links to the 3D Shapes.

...
// remove links to the 3D Shapes
hr = spLinkedRep->RemoveAssociatedReps(l3DShapeList);
...
