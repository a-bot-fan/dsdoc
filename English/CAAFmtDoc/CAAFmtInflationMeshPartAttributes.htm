Inflation Mesh Part Creation

The inflation mesh part can be created by using CATIFmtMeshManager::CreateMeshPart interface method.
A "late type" must be given to indicate which type of mesh part is to be created.
In the case of the inflation mesh part, the late type is "CATFmtInflationMesher".

Inflation Mesh Part Support Definition

The Inflation Mesh Part support can be initialized and managed using the CATIFmtMeshPart interface:

  • SetSupport ( CATFmtPath * iSupport , int iMode = 0 , const CATUnicodeString& iType = "" ) to add a new support to the mesh part
  • RemoveSupport ( CATFmtPath * iSupport = NULL ) to remove a support of the mesh part
  • GetSupport ( CATFmtPath *& oSupport , int iNumber = 1 ) to read a support of the mesh part
  • GetNumberOfSupports () to read the total number of the mesh part supports

Global Mesh Specifications

Global mesh specifications (accept the Geometry specification) can be set or retrieved using CATIFmtMeshPart interface:

  • SetGlobalSpecification ( const CATString &iType , const CATUnicodeString &iName , int iValue ) to set a mesh attribute with an integer value
  • SetGlobalSpecification ( const CATString &iType , const CATUnicodeString &iName , double iValue ) to set a mesh attribute with a double value
  • SetGlobalSpecification ( const CATString &iType , const CATUnicodeString &iName , const CATUnicodeString &iValue ) to set a mesh attribute with a string value
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, int &oValue ) to get a mesh attribute with an integer value
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, double &oValue ) to get a mesh attribute with a double value
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, CATUnicodeString &oValue ) to get a mesh attribute with a string value

iType is the type of specification. iType = "Mesh" for all Global Mesh Specifications.

The Geometry specification can be set, retrieved, or removed using CATIFmtMeshPart interface:

  • SetExternalReference ( const CATString &iType , const CATUnicodeString &iName , CATFmtPath *iSupport ) to set a reference to geomrtic solid support.
  • GetExternalReference ( const CATString &iType , const CATUnicodeString &iName , CATFmtPath *&iSupport ) to retrieve a reference to geomrtic solid support.
  • RemoveExternalReference ( const CATString &iType , const CATUnicodeString &iName , CATFmtPath &*iSupport ) to remove a reference to geometric solid support.

iType is the type of specification and iName is the name of the attribute. iType = "Mesh" and iName = "Geometry"

ElementOrder

Type: int valuated to:

  • 1: to get a mesh with linear hexahedron (HE8) and\or linear wedge (WE6) elements.
  • 2: to get a mesh with quadratic hexahedron (HE20) and\or quadratic wedge (WE15) elements.

default value: 1

This attribute specifies whether linear or quadratic elements will be created.

ThicknessMode

Type: int valuated to:

  • 1: Thickness will be computed automatically either from a solid body associated with a midsurface feature, or if a midsurface feature does not exist the thickness will be derived from a Thin Parts Attribute associated with the part body.
  • 2: Uniform thickness specified using the ThicknessValue parameter.
  • 3: Thickness will be computed automatically from a selected solid body referred by the Geometry parameter.

default value: 1

This attribute specifies the method that will be used to calculate the mesh inflation thickness.

ThicknessValue

Type: double valuated to the inflation thickness This attribute specifies the inflation thickness.
It is only taken into account if Thickness is set to 2.

NbLayers

Type: int valuated to the number of layers to be created. This attribute specifies the number of layers of 3D elements to be created. The layers are uniformly distributed accross the thickness.

Geometry

Type: CATFmtPath valuated to a reference to a solid body. This attribute specifies a reference to a solid body which is used in the calculation of the inflation thickness.
