API's to create the various sensor attributes

Following are the list of Sensor API':

Following in the list of various sensor files along with the API's:

  1. Sensor Base
  2. Sensor
  3. Resultant sensor
  4. Frequency sensor

  • Sensor Base Interface

    interface SMAIAMpaSensorBase:CATIABase

    • 1) PROPERTY Support

    put_Support ( in CATSafeArrayVariant ilsSupport )


    2) PROPERTY SimSupport

    put_SimSupport ( in CATSafeArrayVariant icusSupport )


    3) PROPERTY FrameSelector

    put_FrameSelector ( in SMAIAMpaFramesSelection ispFrameSelector )


    4) Update()


  • Sensor Interface

    interface SMAIAMpaSensor:SMAIAMpaSensorBase

    • 1) PROPERTY ProcessingType

    put_ProcessingType ( in SimProcessingTypes ieProcessingType )

    The following enum values are available for SimProcessingTypes: 

       a) SimScalars 
   b) SimQuantity
   c) SimTensors
   d) SimVectors


    2) PROPERTY VariableType

    put_VariableType( in CATBSTR icsVariableType )


    3) PROPERTY PlyType

    put_PlyType( in CATBSTR icsPlyType )


    4) PROPERTY Location

    put_Location( in SimLocation ieLocation )

    The following enum values are available for SimLocation: 

       a) SimNodes
   b) SimCentroids
   c) SimFaceElements


    5) PROPERTY Averaging

    put_Averaging( in SimAveraging ieAveraging )

    The following enum values are available for SimAveraging: 

       a) SimSectionBoundaries 
   b) SimNoBoundaries


    6) PROPERTY PrincipalDirection

    put_PrincipalDirection(in boolean ibPrincipleDirection )


    7) PROPERTY ComplexValue

    put_ComplexValue( in SimComplexValues ieComplexValue)

    The following enum values are available for SimComplexValues: 

       a) SimValueAtAngle 
   b) SimEnvelopeMaxAbsolute
   c) SimEnvelopeMax 
   d) SimEnvelopeMin
   e) SimPhaseAngle


    8) PROPERTY ComplexValueAtAngle

    put_ComplexValueAtAngle( in double iValueAtAngle )


    9) PROPERTY QuantityType

    put_QuantityType( in CATBSTR icsQuantity )


    10) PROPERTY ComputeQuantityBeforeAveraging

    put_ComputeQuantityBeforeAveraging( in boolean ibQuantityBeforeAveraging )


    11) PROPERTY TransformType

    put_TransformType( in SimTransformTypes ieTransfromType )

    The following enum values are available for SimTransformTypes: 

       a) SimNone 
   b) SimNodal
   c) SimUserDefined


    12) PROPERTY CoordinateSystem

    put_CoordinateSystem( in CATBSTR icsCoordinateSystem )


    13) PROPERTY DeformationEffects

    put_DeformationEffects( in boolean ibDeformationEffects )


    14) SetParameters (in boolean ibMaxFlag, in boolean ibMinFlag, in boolean ibAbsoluteMaxFlag, in boolean ibParameterForEachStepFlag, in boolean ibParameterForEachFrameOrLoadCaseFlag )


  • Resultant Sensor Interface

    interface SMAIAMpaResultantSensor : SMAIAMpaSensorBase

    This interface does not contain any methods. It is derived from SMAIAMpaSensorBase, thus the Sensor Base methods can be used to set the support, frame selector and update the sensor.


  • Frequency Sensor Interface

    interface SMAIAMpaFrequencySensor : SMAIAMpaSensorBase

    This interface does not contain any methods. It is derived from SMAIAMpaSensorBase, thus the Sensor Base methods can be used to set the support, frame selector and update the sensor.
