API's to create the various frame selection attributes

  • Frames Selection Interface

    interface SMAIAMpaFramesSelection : CATIABase


    • Following are the Frame Selection API's:


    1) PROPERTY StepList

    put_StepList ( in CATSafeArrayVariant iStepsList )


    2) PROPERTY SymbolicFrameRange

    put_SymbolicFrameRange ( in CATVariant iSymbolicFrameRange )

    The following values are available for iSymbolicFrameRange. 

      a) SimUndefined                    
  b) SimAllFrames            
  c) SimLastFrameOfEachStep  
  d) SimAllFramesInStep      
  e) SimTimeBasedFrames      
  f) SimFrequencyFrames
