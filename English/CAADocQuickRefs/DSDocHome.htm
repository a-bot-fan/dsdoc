Welcome to the 3DEXPERIENCE Developer Assistance

Designed as a single entry point to all Dassault Systèmes app and brand API documentation, the 3DEXPERIENCE Developer Assistance covers all brands. It can be viewed using Internet Explorer or Mozilla Firefox.

The 3DEXPERIENCE API allows you to create:

  •  C++ batch and interactive applications on top of CATIA, DELMIA, and SIMULIA apps accessing the ENOVIA Collaboration Platform through the ENOVIA Collaborative Environment, excluding the ENOVIA 3DLive standalone solution.

  • Automation macros in VBScript, VBA, C#, and VB.NET.

  • Web and widget applications in JavaScript.

The 3DEXPERIENCE Developer Assistance is made up of technical articles and how-to documentation, use cases along with their code, and reference documentation that can be accessed rapidly via the table of contents on the left. You can also search for a topic, a specific term or phrase directly via the full-text search engine.
