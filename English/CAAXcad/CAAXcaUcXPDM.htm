How to Use MyX-PDM
Use Case

Abstract

This usecase gives you basic information about functionalities of MyX-PDM illustrated by using Sample CAA data. Here it is important to understand this application is only sample application, user can also create his own PDM or use existing PDM or create his own application and replace this application.

Basic Objective

The basic objective of this article is to understand how MyX-PDM menus appear in the application. User will come to know about using MyX-PDM functionalities. Here we have integrated MyX-PDM functionalities in MyX-CAD application.

MyX-PDM Menu

MyX-PDM functionalities are available thourgh MyX-CAD Application. Launch MyX-CAD application from MyX-CAD Launcher. Refer to How to Launch a MyX-CAD [1] for details.

Different functionalities which can be performed with MyX-PDM are listed under MyX-PDM menu in MyX-CAD application.This is shown in [Fig.1].

Fig.1: MyX-PDM Menu

It is necessary to start MyX-PDM before performing any operation. Start MyX-PDM by using MyX-PDM Launcher. Refer to How to Launch a MyX-PDM [2] for details.

When we launch the MyX-PDM the following window appears [Fig.2], In this window, "Install path" is path of .jar files. This files contains the information about bus adapter implementation used in integrating MyX-PDM with the 3DEXPERIENCE platform.

Under"Bus Adapters Found" tag data models which are implemented and respective .jar files present at Install path will be listed. Refer to Sample Bus Adapter for MyX-PDM [3] for details about bus adapter.

"MyX-PDM information area" displays information about ongoing activities performed at MyX-CAD application with MyX-PDM such as login,checkin, checkout etc. It contains information about data used in processing and some details about processes.

Fig.2: MyX-PDM Launcher

MyX-PDM Functionalities

We can perform different operations with MyX-PDM. But before using it, we should connect to MyX-PDM. For this purpose as shown in Fig.1,submenus in MyX-PDM are disabled.

Connect to MyX-PDM:

When user click on "Connect to MyX-PDM" following windows appears [Fig.3] :

Fig.3: Login

Enter the correct user and password and click on Login. Once user is logged in to MyX-PDM with proper credentials , he/she can use other MyX-PDM functionalities. As shown in [Fig.4], Submenus which were disabled get activated.

Fig.4: MyX-PDM Menu

Checkin to MyX-PDM

This functionality allows the user to save the working file in MyX-PDM. For this operation file or structure which is currently opened is used. If no any file is opened then this submenu will be deactivated.

Open the file we want to save, select command "Checkin to MyX-PDM" [Fig.5]

Fig.5: Checkin Menu

 When user click on it, following window will appear[Fig.6]. Enter the name of directory in which we want to save this file.

Fig.6 : Checkin to MyX-PDM

Click on Checkin button,file will be stored in the MyX-PDM in above mentioned directory.

Checkout from MYX-PDM

This functionality allows the user to open a saved file from MyX-PDM database.

Fig.7 : Checkout Menu

When user clicks on "Checkout from MyX-PDM" following windows will appear[Fig.8]. This window contains the list of files which are saved in MyX-PDM.

Fig.8: Checkout from MyX-PDM

Select the file which we want to checkout from the MyX-PDM Database and click on "Checkout". We will get this file opened in in MyX-CAD application.

Disconnect of MYX-PDM:

It will disconnect the MyX-PDM from MyX-CAD application.

When user clicks on this command, following information window will appear [Fig.10].

Fig.10: Information Window of Disconnect

Note : "Send to 3DEXPERIENCE platform" functionality is provided with "X-PDM Addins". Bus adapter and addins are used for implementing it. If bus adapter and addins are not found then these menus will not be visible. Please refer to X-PDM addins[4] for more details.

In ShortIn Short

We have seen the sample PDM ( MyX-PDM) and how different functionalities aggregated under MyX-PDM Menu can be used in MyX-CAD application. This article gives detailed information about how to use MyX-PDM.
