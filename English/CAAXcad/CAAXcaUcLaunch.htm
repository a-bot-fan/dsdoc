Opening the X-CAD Workspace inside Eclipse

For launching the X-CAD Application we have to follow the execution procedure and be more prudent of errors, we need to import the application into Eclipse then launch it from Eclipse so that we can supervise the Eclipse console. Now, we will have a quick look at what to do in this case.

How to launch X-CAD from Eclipse

Import X-CAD into Eclipse
  • Start Eclipse and then enter the path of Workspace repository as you want. The Workspace repository can be any local directory on your machine.
    • Fig.1: Eclipse Workspace Launcher
  • Create a new Java Project in eclipse using File -> New -> Java Project.
  • Copy the content of com folder from each module of the framework and paste it under the src of newly created Java Project.
    • Fig.2: Package Explorer
  • Once packages are ready in eclipse then select all packages present in one module.
    • Fig.3: Package Structure
  • Create jar of this packages using Export.. option.
  • Name of the jar file must be same as that of module name.
    • e.g. Select all packages under MyXCAD.mj and create MyXCAD.jar.
    Note : Create runnable jar for packages under MyXCADLauncher.mj module,it should be named as MyXCADLauncher.jar.
    Also create runnable jar for packages under MyXPDMLauncher.mj module,it should be named as MyXPDMLauncher.jar. Refer to How to launch MyX-PDM [1] for details.
    Fig.4: Export
  • In JAR Export window, select export destination as $Java Project$\bin\win_b64\docs\javaserver\nameofjarfile.
    • Fig.5: JAR Export

Launching the X-CAD Use Case

  • Now In the Project Explorer (Eclipse) Select The package name MyXCADLauncher and 'Right Click' on it. Select "Run As | Java Application".
    • Fig.6: Run The Application

  • After Executing this command we will get the UI for the XCAD Application Fig[7].
    • Fig.7: X-CAD UI
  • User can refer this CAAVIXCADIntegration.edu framework as reference for XML creation and send /receive request to IEF client and generate its own plugin for their own application.
    • Fig.8: Package Structure
    Note: In order to use ENOVIA V6 addin, customer can use packages from CAAVIXCADIntegration.edu.
