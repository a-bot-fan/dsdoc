What's New in 3DEXPERIENCE R2016x Web & Widget Apps Development
Technical Article

Abstract

This article presents Javascript API & associated documentations that are added since 3DEXPERIENCE R2016x GA, compared with the previous release. Some enhancements or maintenance reasons may also require changes regarding API usage or methodologies. This article lists them, and points to the corresponding documentation.
