Define Your Import/Export Dialog Frame
Technical Article

Abstract

This article's main goal is to help partners understand how to define their own frame to manage interactively the importer/exporter parameters.

Create a new import/export dialog frame

To create a new import or export dialog frame, a new C++ class which derives from CATDlgFrame must be created. The class constructor must have an CATDlgDialog input argument which represent the dialog where the frame will be added.

...
CATSimManagerSIMImporterCAAUIFrame::CATSimManagerSIMImporterCAAUIFrame(CATDlgDialog * ipDialog) : CATDlgFrame(ipDialog, "CATSimManagerSIMImporterCAAUIFrame", CATDlgGridLayout)
...

All the dialog components which help the user to define the importer/exporter parameters must be created and initialized in the constructor.

How to plug your frame into the import dialog

To plug the new import dialog frame, a data extension of CATISimManagerSIMImporterUI must be created thanks to CATImplementClass macro on your importer descriptor object.

...
CATImplementClass( CATSimManagerSIMImporterCAAUI, DataExtension, CATISimManagerSIMImporterUI, CATSimManagerSIMImporterCAADescriptor );
CATImplementBOA( CATISimManagerSIMImporterUI, CATSimManagerSIMImporterCAAUI );
...

The implementing class must also derive from CATSimManagerSIMImporterUIAdapter adaptor and have a data member which represents the customized frame.

...
CATSimManagerSIMImporterCAAUI::CATSimManagerSIMImporterCAAUI():CATSimManagerSIMImporterUIAdapter(), _CAAFrame( NULL )
...

Two methods must be implemented : CreateParametersFrame and FillParameterSet.

  • CreateParametersFrame: Creates the frame with UI components dedicated to the importer. The frame's parent dialog is given as input parameter.
  • FillParameterSet: Fills the given parameter set with the parameters defined by the UI components

How to plug your frame into the export dialog

To plug the new export dialog frame, a data extension of CATISimManagerSIMExporterUI must be created thanks to CATImplementClass macro on your exporter descriptor object.

...
CATImplementClass( CATSimManagerSIMExporterCAAUI, DataExtension, CATISimManagerSIMExporterUI, CATSimManagerSIMExporterCAADescriptor );
CATImplementBOA( CATISimManagerSIMExporterUI, CATSimManagerSIMExporterCAAUI );
...

The implementing class must also derive from CATSimManagerSIMExporterUIAdapter adaptor and have a data member which represents the customized frame.

...
CATSimManagerSIMExporterCAAUI::CATSimManagerSIMExporterCAAUI():CATSimManagerSIMExporterUIAdapter(), _CAAFrame( NULL )
...

Two methods must be implemented : CreateParametersFrame and FillParameterSet.

  • CreateParametersFrame: Creates the frame with UI components dedicated to the exporter. The frame's parent dialog is given as input parameter.
  • FillParameterSet: Fills the given parameter set with the parameters defined by the UI components

How to display the import/export dialog frame in 3DExperience session

In the "Structural Model Creation" workbench you can launch the "Import FEM" or "Export FEM" command. Click on the file browser button, and then choose your own import/export format. Choose the correct file and click on OK.

The import dialog has now the more button as shown in the figure just below. By clicking on it, the customized import/export dialog is shown.
