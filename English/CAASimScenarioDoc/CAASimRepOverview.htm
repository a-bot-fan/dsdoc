Scenario and Result representation persistent data model overview
Technical Article

Abstract

This article's main goal is to describe the data model of the Scenario and Result representations. It also allows the partner understanding how he can customize the scenario feature specifications and the generated results with implementing CAA interfaces on features supporting CAA inheritance.
