JavaScript API Reference
The JavaScript API reference documentation is located in the References section. You can access from there:
  • The Widget Class Reference describing the Widget class exposed events, properties, and methods.
  • The list of the frameworks exposing APIs, each of one pointing to the documentation of all its exposed modules, classes, namespaces, etc.
  • All Indexes, and among them, namely:
    • The class index.
    • The class method index.
    • The event index.
    • The function index.
    • The AMD Module and MWeb Module indexes.

    And other useful indexes about JavaScript types.
