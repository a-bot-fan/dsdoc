Glossary

A | D | H | R | T | U | W |

A

AMD
Acronym for Asynchronous Module Definition
Asynchronous Module Definition
A pattern for writing modular JavaScript code. A module and its dependencies can be asynchronously loaded.

D

dashboard
A page browser divided in tab pages. Each tab page can contain severals widget app

H

HTTP
Acronym for HyperText Transfer Protocol.
HTTP verbs
The HTTP commands used by web services as operations, such as GET, PUT, POST, DELETE, HEAD.

R

Run your App
The widget to instantiate your widget in the 3DDashboard.Such a widget is not trusted.

T

Trusted Widget
A Third Party App widget. It gives it the right to access the 3DEXPERIENCE servers, and its resources (js,img,..) are loaded thanks to the proxification.
unTrusted Widget
A Run Your App widget. It prevents the right to access the 3DEXPERIENCE servers, and its resources (js,img,..) are not proxified.

U

URI
Acronym for Uniform Resource Identifier.
URL
Acronym for Uniform Resource Locator.
URN
Acronym for Uniform Resource Name.

W

Widget App
A web application loaded in the 3DDashboard. It must be a UWA Widget.
Widget Platform
An application running on the 3DDashboard server providing services as widget compilation, resources access management, widget/tabpage/dashboard sharing management, ...
