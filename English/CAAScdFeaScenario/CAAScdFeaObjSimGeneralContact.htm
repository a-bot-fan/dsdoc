SimGeneralContact Object
See Also Legend

















Available with the RBM - Simulation Scripting product only.

Use the Add method of the SimFeatures collection to create a new SimGeneralContact object.

Dim cSimFeatures As SimFeatures
...
Dim oSimGeneralContact As SimGeneralContact
Set oSimGeneralContact = cSimFeatures.Add("SimGeneralContact")

Use the GetItem method to return the SimLinkAccess object from its aggregating object, such as the SimContactPropertyAssignment object.

Dim oSimContactPropertyAssignment As SimContactPropertyAssignment
...
Dim oSimLinkAccess As SimLinkAccess
Set oSimLinkAccess = oSimContactPropertyAssignment.GetItem("SimLinkAccess")
