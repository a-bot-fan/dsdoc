Creation Late Types for Scenario Features
Technical Article

Abstract

The following reference article documents the different late types that can be used to create scenario features.

  • Steps Late Types
  • Setup Features Late Types
  • Controls Features Late Types
  • Interactions Late Types
  • Load Features Late Types
  • Restraint Features Late Types
  • Thermal Features Late Types
  • Output Features Late Types
Related Topics
SimFeatures Collection

Steps Late Types

Steps creation is managed by the SimSteps collection by calling the Add method from it.

A "late type" must be given to indicate which type of step is to be created:

Step Late type
Static Step SimStaticStep
Frequency Step SimFrequencyStep
Harmonic Response Step SimHarmonicResponseStep
Modal Dynamic Step SimModalDynamicStep
Static Perturbation Step SimStaticPerturbationStep
Steady State Heat Transfert Step SimSteadyStateHeatTransfertStep
Transient Heat Transfert Step SimTransientHeatTransfertStep
