What You Will Learn With This Use Case

This use case is intended to help you make your first steps in programming with CATIA SFD(Structure Functional Design) Interfaces. Its main intent is to allow you to create member.

The CAASfdCreateMember Use Case

CAASfdCreateMember.m is a use case of the CAAStructureFuncDesignDoc.edu framework that illustrates the CATIA SFD Interfaces framework capabilities.

What Does CAASfdCreateMember Do

The goal of CAASfdCreateMember use case is to show how to create Member.

How to Launch CAASfdCreateMember

To launch CAASfdCreateMember , you will need to set up the build time environment, then compile CAASfdCreateMember along with its prerequisites, set up the run time environment, and then execute the sample.

To launch the use case, execute the following command:

mkrun -c " CAASfdCreateMember Repository Server LoginTicket Title Version"

where:

Repository
Server The server as Host::port_number/RootURI
LoginTicket Defined by one string.
Title Title/V_Name of data to open.
Version Version of the data

* The input model is retrieved by importing the CAASfdCreateMember.3dxml file from the CAAStructureFuncDesign.edu framework.

InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdCGR.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdSR.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdSteel_A42.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdWT18x179_5.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdCreateMember.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This 3D XML file contains the root Product Reference with the following attributes:

  • Title: SfdProduct
  • Version: ---

Where to Find the CAASfdCreateMember Code

The CAASfdCreateMember sample is made of a single class named CAASfdCreateMember located in the CAASfdCreateMember.m module of the CAAStructureFuncDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\CAASfdCreateMember.m\

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATAdpOpenSaveServices Class to manage Open and Save in session.
CATISfdUseFunctionFactory Interface to create SFD Panel/Member.
CATISfdUseMember Interface dedicated to SFD member.
CATIStrUseCategoryMngt Interface to set/get category.
CATIStrUseProfileSurfSurf Interface for set/get member support.
CATIStrUseProfileLimitMngt Interface dedicated to set/get limits for member.
CATIStrUseMaterialMngt Interface to set/get material.
CATIStrUseProfileType Interface to set/get profile type.
CATIStrUseSectionMngt Interface to set/get section attributes e.g. section name , anchor point.
