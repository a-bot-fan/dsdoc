What You Will Learn With This Use Case

This use case is intended to help you make your first steps in programming with CATIA SFD(Structure Functional Design) Interfaces. Its main intent is to allow you to create plate.

The CAASfdCreatePlate Use Case

CAASfdCreatePlate.m is a use case of the CAAStructureFuncDesignDoc.edu framework that illustrates the CATIA SFD Interfaces framework capabilities.

What Does CAASfdCreatePlate Do

The goal of CAASfdCreatePlate use case is to show how to create plate.

How to Launch CAASfdCreatePlate

To launch CAASfdCreatePlate , you will need to set up the build time environment, then compile CAASfdCreatePlate along with its prerequisites, set up the run time environment, and then execute the sample.

To launch the use case, execute the following command:

mktun -c " CAASfdCreatePlate Repository Server Login_Ticket Title Version"

where:

Repository PLM1
Server The server as Host::port_number/RootURI
Login_Ticket defined by one string
Title The value of the Title attribute of the PLM representation reference containing the data to create the user feature reference.
Version Version

* The input model is retrieved by importing the CAASfdCreatePlate.3dxml file from the CAAStructureFuncDesign.edu framework.

InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdCGR.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdSR.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdSteel_A42.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdCreatePlate.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This 3D XML file contains the root Product Reference with the following attributes:

  • Title: SfdProduct
  • Version: ---

Where to Find the CAASfdCreatePlate Code

The CAASfdCreatePlate sample is made of a single class named CAASfdCreatePlate located in the CAASfdCreatePlate.m module of the CAAStructureFuncDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\CAASfdCreatePlate.m\

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATAdpOpenSaveServices Class to manage Open and Save in session.
CATISfdUseFunctionFactory Interface to create SFD Panel/Member.
CATISfdUsePanel Interface dedicated to SFD panel.
CATIStrUseCategoryMngt Interface to set/get category.
CATIStrUsePanelSurf Interface for set/get panel support.
CATIStrUsePanelLimitMngt Interface dedicated to set/get limits for panel.
CATIStrUseMaterialMngt Interface to set/get material.
CATIStrUsePlateExtrusionMngt Interface to set/get plate geometry attributes.
CATISfdUsePlatesMngt Interface to create split plates.
