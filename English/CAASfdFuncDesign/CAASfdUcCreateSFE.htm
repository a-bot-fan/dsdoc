What You Will Learn With This Use Case

This use case is intended to help you make your first steps in programming with CATIA SFD(Structure Functional Design) Interfaces. Its main intent is to allow you to create StiffenerOnFreeEdge.

The CAASfdCreateSFE Use Case

CAASfdCreateSFE.m is a use case of the CAAStructureFuncDesignDoc.edu framework that illustrates the CATIA SFD Interfaces framework capabilities.

What Does CAASfdCreateSFE Do

The goal of CAASfdCreateSFE use case is to show how to create StiffenerOnFreeEdge.

How to Launch CAASfdCreateSFE

To launch CAASfdCreateSFE , you will need to set up the build time environment, then compile CAASfdCreateSFE along with its prerequisites, set up the run time environment, and then execute the sample.

To launch the use case, execute the following command:

mkrun -c " CAASfdCreateSFE Repository Server Login_Ticket Title Version"

where:

Repository
Server The server as Host::port_number/RootURI
Login_Ticket defined by one string
Title The value of the Title attribute of the PLM representation reference containing the data to create the user feature reference.
Version Version

* The input model is retrieved by importing the CAASfdCreateSFE.3dxml file from the CAAStructureFuncDesign.edu framework.

InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdCGR.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdSR.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdSteel_A42.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdWT18x179_5.3dxml
InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\InputData\CAASfdCreateSFE.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This 3D XML file contains the root Product Reference with the following attributes:

  • Title: SfdProduct
  • Version: ---

Where to Find the CAASfdCreateSFE Code

The CAASfdCreateSFE sample is made of a single class named CAASfdCreateSFE located in the CAASfdCreateSFE.m module of the CAAStructureFuncDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureFuncDesign.edu\CAASfdCreateSFE.m\

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATAdpOpenSaveServices Class to manage Open and Save in session.
CATIStrUseStiffenerMngt Interface to manage StiffenerOnFreeEdge.
CATIStrUseProfileOnOpening Interface for set/get StiffenerOnFreeEdge support opening.
CATIStrUseMaterialMngt Interface to set/get material.
CATIStrUseProfileType Interface to set/get profile type.
CATIStrUseSectionMngt Interface to set/get section attributes e.g. section name , anchor point.
