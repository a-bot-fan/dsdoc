CXPARSettings Setting Repository
Quick Reference

CXPARSettings

Role: Setting to handle CXP model parameters

The CXPARSettings repository has the following characteristics:

  • Primary Tab Page: /Interactive 3DEXPERIENCE/Creative Experience/Augmented Reality
  • Repository File: CXPARSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CXPARSettings setting repository attributes:

  • CustoIntrinsicParms
    Role: Parameter to set custom intrinsic parameters of the webcam
    • Type: string
    • Default value: xsize ysize cc_x cc_y fc_x fc_y k1 k2 p1 p2 k3 0 10
  • CustoStreamName
    Role: Parameter to set a custom webcam stream name
    • Type: string
    • Default value: Logitech HD Pro Webcam C910 RGB24 640x480
  • FriendlyName
    Role: Parameter to set the webcam friendly name
    • Type: string
    • Default value: Logitech HD Pro C910 - 480p
  • HideMarkerDuringPlay
    Role: Parameter to set hide marker during play
    • Type: boolean
    • Default value: false
  • IntrinsicParms
    Role: Parameter to set intrinsic parameters of the webcam
    • Type: string
    • Default value: 640 480 322.038986554 247.815613213 545.551353689 546.090030722 0.00332661127594 -0.0323724273914 -0.000625287459287 0.00212702830458 0 0 10
  • StreamName
    Role: Parameter to set the webcam stream name
    • Type: string
    • Default value: Logitech HD Pro Webcam C910 RGB24 640x480
  • TrackingAccuracy
    Role: Tracking accuracy of augmented reality techno: 0 is fast but not accurate, 5 is slow but accurated
    • Type: integer
    • Default value: 5
    • Min value: 0
    • Max value: 5
  • WebcamChannel
    Role: Webcam channel id
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
