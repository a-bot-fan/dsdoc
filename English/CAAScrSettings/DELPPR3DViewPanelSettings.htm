DELPPR3DViewPanelSettings Setting Repository
Quick Reference

DELPPR3DViewPanelSettings

Role: Settings to handle the parameters of 3DView Panel

Detailed role:This setting allows to customize the appearance and the behavior of 3DView Panel

The DELPPR3DViewPanelSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/System Product BuildUp
  • Repository File: DELPPR3DViewPanelSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELPPR3DViewPanelSettings setting repository attributes:

  • IsDisplayedWithColorIn3DViewPanel
    Role: Color Mode in 3DView Panel
    • Type: integer
    • Default value: 0
  • ReframeOnIn3DViewPanel
    Role: Reframe 3DView Panel when modifying the selection or options
    • Type: integer
    • Default value: 1
  • ShowAssignedResourcesIn3DViewPanel
    Role: Display assigned resources in 3DView Panel
    • Type: integer
    • Default value: 0
  • ShowOnlyConcernedPartsIn3DViewPanel
    Role: Display only assigned parts in 3DView Panel
    • Type: integer
    • Default value: 0
