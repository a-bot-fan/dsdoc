GeneralAudio Setting Repository
Quick Reference

GeneralAudio

Role: Option to handle Audio Capture parameters

The GeneralAudio repository has the following characteristics:

  • Primary Tab Page: /General/Audio
  • Repository File: GeneralAudio
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GeneralAudio setting repository attributes:

  • BitsPerSample
    Role: Parameter to set bits per sample used during audio capture
    • Type: integer
    • Default value: 1
    • Values:
      0 8-bit
      1 16-bit
  • CaptureParams
    Role: Parameter to set sample rate/bits per sample/channel count used during audio capture
    • Type: unsigned integer
    • Default value: 4
    • Min value: 0
    • Max value: 10
    • Step: 1
  • ChannelCount
    Role: Parameter to set channel count used during audio capture
    • Type: integer
    • Default value: 1
    • Values:
      0 Mono
      1 Stereo
  • SampleRate
    Role: Parameter to set sample rate used during audio capture
    • Type: integer
    • Default value: 2
    • Values:
      0 8000Hz
      1 11025Hz
      2 22050Hz
      3 44100Hz
      4 48000Hz
