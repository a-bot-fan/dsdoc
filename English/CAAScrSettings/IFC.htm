IFC Setting Repository
Quick Reference

IFC

Role:

The IFC repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/IFC
  • Repository File: IFC
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

IFC setting repository attributes:

  • Application Protocol
    Role: Type of Application Protocole
    • Alias: IFCReleaseVersion
    • Type: short integer
    • Default value: 0
    • Values:
      0 IFC 2x3
      1 IFC 2x4
  • Exact geometry as
    Role: Excat geometry as
    • Alias: ExportRepMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Export exact geometry on IFC file
      1 Export tessellated geometry on IFC file
  • Export also NoShow entites
    Role: Enables export of the not shown entities.
    • Alias: ShowMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Prefer 3D tessellated geometry
    Role: The import chose this one if it exists on IFC files.
    • Alias: ImportTessPreferedMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
