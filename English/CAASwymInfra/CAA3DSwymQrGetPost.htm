Get Post

Role

Returns informations about an existing post.

Request

 Method  Path 
POST api/post/get

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
community_post_id yes A positive number. The identifier inside the community of the post to get.
Container for all post parameters.
request_id yes 0 For internal use.
