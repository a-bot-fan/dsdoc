List Medias

Role

Returns a list of the media belonging to a community.

Request

 Method  Path 
POST api/media/list

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
community_id yes A positive number. Identifier of the community where the media will be listed from.
limit no A number between 1 and 128. Default value if not indicated is 10. The maximum number of elements returned by the list.
page no A number greater than one. Default value if not indicated is 1. Optional parameter to be used with "limit". For paginated result (see Paginated Result).
Container for all POST parameters.
request_id yes 0 For internal use.

Response

The response is a standard 3DSwym JSON (see JSON as Output) where the key result is an array. Each element of the array is a JSON composed as follows:
