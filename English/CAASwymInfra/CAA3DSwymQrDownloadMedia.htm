Download Media

Role

Downloads a media or a thumbnail of a media.

Request

 Method  Path 
GET api/media/stream

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
id yes A positive number. Identifier of the media to download.
key no A string. Possible values are:
  • s_thumb: small size.
  • m_thumb: medium size.
  • l_thumb: large size.
 Mandatory parameter to be used with type "thumb". Desired size of the thumbnail.
type yes A string. Possible values are:
  • thumb: to get a pre-processed thumbnail ; this parameter must be used with an additional "key" parameter.
  • original: download the original media file (the media uploader may have deactivated this feature).
 Type of download for the thumbnail.

Response

The response is an image (see Image as Output).
