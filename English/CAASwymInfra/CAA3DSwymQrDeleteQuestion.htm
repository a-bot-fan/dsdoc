Delete Question

Role

Deletes one or more iQuestion inside a community.

Request

 Method  Path 
POST api/question/delete

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
community_id yes A positive number. Identifier of the community where the iQuestions will be deleted.
iquestionIds yes An array of positive numbers. Array of iQuestion identifiers inside the community.
Container for all post parameters.
request_id yes 0 For internal use.

Response

The response is a standard 3DSwym JSON (see JSON as Output) where the key result is itself a JSON composed as follows:

 Key   Description   Value   Applicability 
community_id Community identifier of the deleted post. A positive number. yes
