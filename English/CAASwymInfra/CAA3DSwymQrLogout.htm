Logout

Role

Logs user out of 3DSwym. If user is not logged in or already logged out, it redirects to 3DSwym login page.

Request

 Method  Path 
GET api/passport/logout

Parameters

Parameters

Name Required Value Description
service NO An URL service. The format is URL-encoded. The browser might be automatically redirected to the service URL after the log out is performed by the CAS server. If this service URL is not specified, a notification message can be displayed (it depends on the service provider implementation)

Response

The response is an HTML page which content may vary depending on input parameters and request result.

Example

Example of trace file

##CLIENT REQUEST
##------------------------------------------------------------
