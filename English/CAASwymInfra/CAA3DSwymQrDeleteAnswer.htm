Delete Answer

Role

Delete an answer of an iQuestion.

Request

 Method  Path 
POST api/answer/delete

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
id yes A positive number. Identifier of the answer to delete.
Container for all POST parameters.
request_id yes 0 For internal use.

Response

The response is a JSON composed as follows:

 Key   Description   Value   Applicability 
monitoring no
nb_result Status of the deletion. A boolean:
  • true: the item is deleted.
  • false: the item is not deleted.
yes
request_id no
result Information about the deletion.
 Key   Description   Value   Applicability 
id Identifier of the deleted answer. A positive number. yes
moderation_status Status of the deleted answer. A string. Possible values are:
  • no_moderation: comment has not been changed.
  • updated: updated by a community owner or a platform administrator.
  • deleted: deleted by a community owner or a platform administrator.
  • updated_by_author: updated by the author of the content.
  • deleted_by_author: deleted by the author of the content.
 yes
yes
