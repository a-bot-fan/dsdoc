List Who Endorse

Role

Returns a list of user who endorsed the content specified in input.

Request

 Method  Path 
GET api/user/getwhoendorsed

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
limit no A number greater than one. The maximum number of elements returned by the list.
model yes A string. Possible values are:
  • Community: for a community.
  • CommunityPost: for a blog post.
  • Media: for a media.
  • QnaQuestion: for an iQuestion.
  • Dsi_DFSTree_WikiTree: for a Wiki page.
  • Dsi_Ideation_Ideation: for Ideation.
 Type of the data to endorse.
model_id yes A positive number. Identifier of the data to endorse.
start no A positive number. The start index for the results. If not set the default value 0 is used. It replaces the attribute "page" for this specific web service.

Response

The response is a standard 3DSwym JSON (see JSON as Output) where the key result is an array. Each element of the array is a JSON composed as follows:

 Key   Description   Value   Applicability 
first_name First name. A string. An empty string if information is not shared. yes
hash_key Crypted identifier. A hash key. yes
last_name Last name. A string. yes
nickname no
