Edit Media Comment

Role

Edits a existing comment of a media. Returns the edited media.

Request

 Method  Path 
POST api/media/editcomment

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
comment yes A string. Format is HTML-like (see Format HTML-like). Edited message of the comment.
id yes A positive number. Identifier of the comment to edit.
Container for all POST parameters.
request_id yes 0 For internal use.

Response

The response is a standard 3DSwym JSON (see JSON as Output) where the key result is itself a JSON composed as follows:

 Key   Description   Value   Applicability 
comment The edited comment text. A string. Format is HTML-like (see Format HTML-like). yes
id Comment identifier. A positive number. yes
