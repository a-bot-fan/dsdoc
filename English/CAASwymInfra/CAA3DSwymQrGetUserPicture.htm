Get User Picture

Role

Gets a user profile picture. Returns a default picture if user has not uploaded any picture. The user is identified either by a login name or a hash key value.

Request

 Method  Path 
GET api/user/getpicture

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
format no A string. Possible values are:
  • umini: (ultra mini) returns a PNG picture, size: 15x15 pixels.
  • mini: returns a PNG picture, size: 34x34.
  • normal: returns a PNG picture, size: 123x123. Used as default if parameter has not been set.
 Format of the picture.
hash_key no A hash key. Hash key of the user to get profile image from. Can be used instead of login.
login no A string. Login (equivalent to username) of the user to get profile image from.

Response

The response is an image (see Image as Output).
