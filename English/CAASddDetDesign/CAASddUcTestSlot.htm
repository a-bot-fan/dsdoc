What You Will Learn With This Use Case

This use case is intended to help you make your first steps in programming with CATIA structure Interfaces. Its main intent is to allow you to retrieve Slot information.

The CAASddTestSlot Use Case

CAASddTestSlot.m is a use case of the CAAStructureDetDesignDoc.edu framework that illustrates the CATIA SDD Interfaces framework capabilities.

What Does CAASddTestSlot Do

The goal of CAASddTestSlot use case is to show how to retrieve Slot information.

How to Launch CAASddTestSlot

To launch CAASddTestSlot , you will need to set up the build time environment, then compile CAASddTestSlot along with its prerequisites, set up the run time environment, and then execute the sample.

To launch the use case, execute the following command:

mkrun -c " CAASddTestSlot Repository Server Login_ticket Title"

where:

Repository
Server The server as Host::port_number/RootURI
Login_Ticket The login ticket to connect to the base.
Title The value of the Title attribute of the PLM representation reference containing the data to create the user feature reference.

* The input model is retrieved by importing the CAASddTestSlot.3dxml file from the CAAStructureDetDesign.edu framework.

InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddCGR.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddSR.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddSteel_A42.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddWT18x179_5.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\RECT_UNI_WT.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddTestSlot.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This 3D XML file contains the root Product Reference with the following attributes:

  • Title: SddProduct
  • Version: ---

Where to Find the CAASddTestSlot Code

The CAASddTestSlot sample is made of a single class named CAASddTestSlot located in the CAASddTestSlot.m module of the CAAStructureDetDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddTestSlot.m\

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATAdpOpenSaveServices Class to manage Open and Save in session.
CATISddUseProductPlate Class to retrieve the SddPlate.
CATISddUsePlate Interfaces dedicated to SddPlate.
CATIStrUseSlotMngt Interfaces dedicated to manage Slot.
CATIStrUseSlot Interface dedicated to Slot.
CATIStrUseDetailFeature Interfaces provided to manage slot parameters and slot info.

// - Get SddProductPlate

  if(!!spRootProduct)
  {
    CATListPtrCATIPLMNavOccurrence listOfChildrenOccurrences;
    if (SUCCEEDED(spRootProduct->ListChildren(listOfChildrenOccurrences)))
    {
      int i, isize = listOfChildrenOccurrences.Size();
      for (i = 1 ; i <=isize ; ++i)

        CATIPLMNavOccurrence* &piCurrentOccurrence =listOfChildrenOccurrences[i];

        if (piCurrentOccurrence) piCurrentOccurrence->Release(); piCurrentOccurrence = NULL;

  }
