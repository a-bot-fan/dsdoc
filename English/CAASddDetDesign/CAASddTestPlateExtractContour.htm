What You Will Learn With This Use Case

This use case is intended to help you make your first steps in programming with CATIA structure Interfaces. Its main intent is to allow you to retrieve different shapes related to a given SDD plate.

The CAASddTestPlateExtractContour Use Case

CAASddTestPlateExtractContour.m is a use case of the CAAStructureDetDesignDoc.edu framework that illustrates the CATIA SDD Interfaces framework capabilities.

What Does CAASddTestPlateExtractContour Do

The goal of CAASddTestPlateExtractContour use case is to show how to retrieve information related todifferent shapes related to a given SDD plate.

This component:

  • Navigates to a desired product in the SpecTree
  • Navigates and selects a product Plate.
  • Extracts information related to different shapes related to the Plate.

How to Launch CAASddTestPlateExtractContour

To launch CAASddTestPlateExtractContour , you will need to set up the build time environment, then compile CAASddTestPlateExtractContour along with its prerequisites, set up the run time environment, and then execute the sample.

To launch the use case, execute the following command:

mkrun -c " CAASddTestPlateExtractContour Repository Server Login_Ticket Title"

where:

Repository PLM1
Server The server as Host::port_number/RootURI
Login_Ticket defined by one string
Title The value of the Title attribute of the PLM representation reference containing the data to create the user feature reference.

* The input model is retrieved by importing the CAASddTestPlate.3dxml file from the CAAStructureDetDesign.edu framework.

InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddCGR.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddSR.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddSteel_A42.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddTestPlate.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This 3D XML file contains the root Product Reference with the following attributes:

  • Title: SddProduct
  • Version: ---

Where to Find the CAASddTestPlateExtractContour Code

The CAASddTestPlateExtractContour sample is made of a single class named CAASddTestPlateExtractContour located in the CAASddTestPlateExtractContour.m module of the CAAStructureDetDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddTestPlateExtractContour.m\

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATAdpOpenSaveServices Class to manage Open and Save in session.
CATISddUseProductPlate Class to retrieve the SddPlate.
CATIStrUsePanelLimitMngt Interface to manage the limits of SFD panel, SDD panel and SDD plate.It is used to retrieve the SDD Plate's LimitType
CATISddUsePlateContourExtraction Interfaces dedicated to retrieve different shapes related to a given SDD plate.
