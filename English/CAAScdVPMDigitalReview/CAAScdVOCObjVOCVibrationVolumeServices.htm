The VOCVibrationVolumeServices object lets you create a Vibration Volume from a Reference.

Retrieving the VOCVibrationVolumeServices Object

To retrieve the VOCVibrationVolumeServices Object, you need to call the Service from the Representation Reference of a Part.

Dim myPart As Part 
Set myPart = oEditor.ActiveObject 
Dim MyVPMRepRef As VPMRepReference 
Set MyVPMRepRef = myPart.Parent 

Dim myVOCVibVolServices ' As VOCVibrationVolumeServices 
Set myVOCVibVolServices = MyVPMRepRef.GetItem("VOCVibrationVolumeServices")

 

Using the VOCVibrationVolumeServices Object

Use the VOCVibrationVolumeServices object to create Vibration Volumes with a position file or with a track.

  • With a position file : 
    • ' Gets the position File 
Dim vocVibrationTest As String 
vocVibrationTest = "filepath/PositionFile.txt" 

' Gets the VOC Vibration Volume Services 
Dim myVOCServices ' As VOCVibrationVolumeServices 
Set myVOCServices = MyVPMRepRef.GetItem("VOCVibrationVolumeServices") 

myVOCServices.CreateVibrationVolumes MyInputProducts, ObjVPMReference, vocVibrationTest, 2#

    To get myVPMRepRef, MyInputProducts and ObjVPMReference see the VOCServices Object.

    The position File can be a .txt file (see sample), a .VDA or a .mvf.
