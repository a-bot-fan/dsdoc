Drafting2DLUseItf Interface CATI2DLUIFilterServices

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATI2DLUIFilterServices

This interface of services ease creation of filterable elements that describe a sub-assembly.

Method Index


o CreateFilterableFromPath(CATPathElement*,CATPathElement*)
Creates valid object for filter for Assembly pointing from a given CATPathElement.
o CreatePathFromFilterable(CATPathElement*,CATBaseUnknown*)
Creates CATPathElement for visualization from iUIActivPath (Path of UIActive Object) and a filterable (CATI2DLVisuFilterable).

Compares 2 Filterables : are they equal or not.
o IsPathFilterable(CATPathElement*,CATPathElement*)
Tests if a CATPathElement is a valid path to create a filterable object in a Visu Filter.

o IsPathFilterable
public virtual IsPathFilterable( iUIActivePath,
iInstancePath)
Tests if a CATPathElement is a valid path to create a filterable object in a Visu Filter. Sample of code : ... // retrieving UIActive Path in command Editor (no check on pointers is shown in this sample) CATFrmEditor *pEditor = GetEditor(); CATPathElement UIActivePath = pEditor->GetUIActiveObject(); // retrieving selected object in a Selection Agent (both prevaluation or valuation state are possible) CATPathElement *selectedPath = _mySelectionAgent->GetValue(); // checking selected path validity CATI2DLUIFilterServices_var spFilterManager = CAT2DLUIFilterServices::CreateFilterServices(); if (S_OK == spFilterManager->IsPathFilterable(&UIActivePath,selectedPath)) { // adding this sub-assembly in a Displaying Filter CATBaseUnknown_var hFilterable=spFilterManager->CreateFilterableFromPath(&UIActivePath,selectedPath); _myFilter -> SetFilterInfo( hFilterable, CATIDibVisuFilterAccess::VISIBLE, TRUE ) ) ) } ...
Parameters:
iUIActivePath
[in]
iInstancePath
[in]
Returns:
An HRESULT value.
Legal values:
S_OK
Path is Valid
S_FALSE
Path is NOT Valid
E_FAIL
Method FAILED
