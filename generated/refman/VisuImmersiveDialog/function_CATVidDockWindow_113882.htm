VisuImmersiveDialog Global Function CATVidDockWindow


CATVidDockWindow( i_pViewer,
const i_windowName,
i_side)
Docks the window identified by the given name in the given viewer at the expected docking side.
Parameters:
i_pViewer
the viewer containing the window If the passed i_pViewer is NULL, this method will search in all the viewers.
i_windowName
the window name
i_side
the docking side 0 : left 1 : top 2 : right 3 : bottom
Returns:
S_OK if the window has been retrieved and succesfully docked, E_FAIL otherwise.
