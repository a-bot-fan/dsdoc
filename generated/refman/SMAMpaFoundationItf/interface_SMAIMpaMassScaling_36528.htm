Enumerated Types


o MassScalingBehavior 
enum MassScalingBehavior {
  BeginningOfStep,
  ThroughoutStep,
  ResetMassMatrix
}
The mass scaling behavior.
Parameters:
BeginningOfStep
The mass will be scaled once at the beginning of the step for which it is specified.
ThroughoutStep
The mass will be scaled at the beginning of a step and periodically during that step. Mass scaling factors will be calculated automatically and applied, as required, throughout the step.
ResetMassMatrix
Discontinue all of the mass scaling methods of a prior step.
o MassScalingMethod 
enum MassScalingMethod {
  Uniform,
  BelowMin,
  SameTimeIncrement
}
The mass scaling method.
Parameters:
Uniform
Masses of the elements are scaled equally so that the smallest element stable time increment of the scaled elements equals the specified target.
BelowMin
Masses of only the elements whose element stable time increments are less than the specified target are scaled. The masses of these elements will be scaled so that the element stable time increments equal the specified target.
SameTimeIncrement
Masses of all elements are scaled so that they all have the same element stable time increment equal to the specified target.
