o CreateConnectionProperty
public virtual CreateConnectionProperty( const iPropertyType,
const ispFemRep,
ospConnectionProperty)
Creates a connection property using the specified engineering connection.
Parameters:
iPropertyType
[in] The type of connection property to be created.
Possible values are :
  • SMAMpaConnector : Creates a connector property. It adheres to the SMAIMpaConnector interface.
  • SMAMpaRigidConnection : Creates a rigid connection property. It adheres to the SMAIMpaRigidConnection interface.
  • SMAMpaSpring : Creates a spring connection property. It adheres to the SMAIMpaSpring interface.
  • SMAMpaTie : Creates a tie connection property. It adheres to the SMAIMpaTie interface.
  • SMAMpaVirtualBolt : Creates a virtual bolt connection property. It adheres to the SMAIMpaVirtualBolt interface.
  • SMAMpaCoupling : Creates a coupling connection property. It adheres to the SMAIMpaCoupling interface.
  • SMAMpaPointFastener : Creates a point fastener connection property. It adheres to the SMAIMpaPointFastener interface.
  • SMAMpaLineFastener : Creates a line fastener connection property. It adheres to the SMAIMpaLineFastener interface.
To avoid update warning message, the connection property should be created under a compatible engineering connection. The list of compatible engineering connection types for different connection properties is as following:
  • For SMAMpaConnector : Cylindrical, Free, Revolute, Rigid or Spherical.
  • For SMAMpaRigidConnection : Cylindrical, Free, Revolute, Rigid or Spherical.
  • For SMAMpaSpring : Cylindrical, Free, Planar, Prismatic, Revolute, Rigid or Spherical.
  • For SMAMpaTie : Free, Planar or Rigid.
  • For SMAMpaVirtualBolt : Free or Rigid.
  • For SMAMpaCoupling : Cylindrical, Free, Planar, Prismatic, Revolute, Rigid or Spherical.
  • For SMAMpaPointFastener : Free or Rigid.
  • For SMAMpaLineFastener : Free or Rigid.
ispFEMRep
[in] The FEM representation under which to create the connection property mesh part.
ospConnectionProperty
[out] The created connection property.
Returns:
S_OK if successful.
