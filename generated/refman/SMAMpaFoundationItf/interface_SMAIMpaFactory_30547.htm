SMAMpaFoundationItf Interface SMAIMpaFactory

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface SMAIMpaFactory

Interface to create and remove simulation features.

Methods


o CreateFeature
public virtual CreateFeature( const iFeatureType,
ospFeature)
Creates a feature.
Parameters:
iFeatureType
[in] The type of the feature to be created. Possible values are :
  • For Properties set :
    • SMAMpaSolidSection : Creates a solid section feature. It adheres to the SMAIFeaSolideSection interface.
    • SMAMpaFluidSection : Creates a fluid section feature. It adheres to the SMAICfdFluidProperty interface.
    • SMAMpaGasketProperty : Creates a gasket section feature. It adheres to the SMAIMpaGasketProperty interface.
    • SMAMpaBeamSection : Creates a beam section feature. It adheres to the SMAIMpaBeamSection interface.
    • SMAMpaContinuumShellSection : Creates a continuum shell section feature. It adheres to the SMAIMpaContinuumShellSection interface.
    • SMAMpaShellSection : Creates a shell section feature. It adheres to the SMAIMpaShellSection interface.
    • SMAMpaCompositeShellSection : Creates a composite shell section feature. It adheres to the SMAIMpaCompositeShellSection interface.
    • SMAMpaMeshedBolt : Creates a mesh bolt feature. It adheres to the SMAIMpaMeshedBolt interface.
  • For Abstractions set :
    • SMAMpaAcousticAbsorber : Creates an acoustic absorber feature. It adheres to SMAIMpaAcousticAbsorber interface.
    • SMAMpaCyclicSymmetry : Creates an analytic rigid surface feature. It adheres to SMAIMpaCyclicSymmetry interface.
    • SMAMpaAnalyticRigidSurface : Creates an analytic rigid surface feature. It adheres to SMAIMpaAnalyticRigidSurface interface.
    • SMAMpaRigidBodyConstraint : Creates a rigid body constraint feature. It adheres to SMAIMpaRigidBodyConstraint interface.
    • SMAMpaMassPerArea : Creates a nonstructural mass per area feature. It adheres to SMAIMpaNonstructuralMass interface.
    • SMAMpaMassPerLength : Creates a nonstructural mass per length feature. It adheres to SMAIMpaNonstructuralMass interface.
    • SMAMpaMassPerVolume : Creates a nonstructural mass per volume feature. It adheres to SMAIMpaNonstructuralMass interface.
    • SMAMpaMassRotaryInertia : Creates a mass rotary inertia feature. It adheres to SMAIMpaMassRotaryInertia interface.
    • SMAMpaSurfaceFluidCavity : Creates a surface fluid cavity feature. It adheres to SMAIMpaSurfaceFluidCavity interface.
    • SMAMpaAcousticCoupling : Creates an acoustic coupling feature. It adheres to SMAIMpaAcousticCoupling interface.
  • For Behaviors set :
    • SMAMpaBeamProfile : Creates a beam profile feature. It adheres to the SMAIMpaBeamProfile interface.
  • For Steps :
    • SMAMpaStaticStep : Creates a static step. It adheres to SMAIMpaStaticStep interface.
    • SMAMpaFrequencyStep : Creates a frequency step. It adheres to SMAIMpaFrequencyStep interface.
    • SMAMpaComplexFrequencyStep : Creates a complex frequency step. It adheres to SMAIMpaComplexFrequencyStep interface.
    • SMAMpaHarmonicResponseStep : Creates an harmonic response step. It adheres to SMAIMpaHarmonicResponseStep interface.
    • SMAMpaModalDynamicStep : Creates an modal dynamic step. It adheres to SMAIMpaModalDynamicStep interface.
    • SMAMpaStaticPerturbationStep : Creates a static perturbation step. It adheres to SMAIMpaStaticPerturbationStep interface.
    • SMAMpaSteadyStateHeatTransferStep : Creates a steady state heat transfer step. It adheres to SMAIMpaSteadyStateHeatTransferStep interface.
    • SMAMpaTransientHeatTransferStep : Creates a transient heat transfer step. It adheres to SMAIMpaTransientHeatTransferStep interface.
    The SMAIMpaScenarioStep::CreateElementTypeAssignment and SMAIMpaScenarioStep::CreateAnalysisDefaultOutputRequests methods must be called after the first step is created.
  • For Setup Features :
    • SMAMpaTabularAmplitude : Creates a tabular amplitude feature. It adheres to SMAIMpaTabularAmplitude interface.
  • For Controls Features :
    • SMAMpaStructuralDamping : Creates a damping feature. It adheres to SMAIMpaStructuralDamping interface.
    • SMAMpaMassScaling : Creates a mass scaling feature. It adheres to SMAIMpaMassScaling interface.
  • For Interaction Features :
    • SMAMpaContactControls : Creates a contact controls feature. It adheres to SMAIMpaContactControls interface.
    • SMAMpaContactInterference : Creates a contact interference feature. It adheres to SMAIMpaContactInterference interface.
    • SMAMpaContactInitialization : Creates a contact interaction initialization feature. It adheres to SMAIMpaContactInitialization interface.
    • SMAMpaSurfaceBasedContactProperty : Creates a contact interaction property feature. It adheres to SMAIMpaSurfaceBasedContactProperty interface.
    • SMAMpaGeneralContact : Creates a general contact feature. It adheres to SMAIMpaGeneralContact interface.
    • SMAMpaSurfaceBasedContact : Creates a surface-based contact feature. It adheres to SMAIMpaSurfaceBasedContact interface.
  • For Load Features :
    • SMAMpaAppliedRotation : Creates an applied rotation. It adheres to SMAIMpaAppliedRotation interface.
    • SMAMpaAppliedTranslation : Creates an applied rotation. It adheres to SMAIMpaAppliedTranslation interface.
    • SMAMpaBoltDisplacement : Creates a bolt displacement. It adheres to SMAIMpaBoltDisplacement interface.
    • SMAMpaBoltForce : Creates a bolt force. It adheres to SMAIMpaBoltForce interface.
    • SMAMpaBoltTorque : Creates a bolt torque. It adheres to SMAIMpaBoltTorque interface.
    • SMAMpaForce : Creates a force. It adheres to SMAIMpaForce interface.
    • SMAMpaImportedForce : Creates a force. It adheres to SMAIMpaImportedForce interface.
    • SMAMpaPressure : Creates a pressure. It adheres to SMAIMpaPressure interface.
    • SMAMpaRemoteForce : Creates a remote force. It adheres to SMAIMpaRemoteForce interface.
    • SMAMpaRemoteTorque : Creates a remote torque. It adheres to SMAIMpaRemoteTorque interface.
    • SMAMpaTorque : Creates a torque. It adheres to SMAIMpaTorque interface.
    • SMAMpaInertiaRelief : Creates a inertia relief. It adheres to SMAIMpaInertiaRelief interface.
    • SMAMpaConnectorForce : Creates a connector force. It adheres to SMAIMpaConnectorForce interface.
    • SMAMpaConnectorMoment : Creates a connector moment. It adheres to SMAIMpaConnectorMoment interface.
  • For Restraint Features :
    • SMAMpaBoltRestraint : Creates a bolt restraint. It adheres to SMAIMpaBoltRestraint interface.
    • SMAMpaClamp : Creates a clamp. It adheres to SMAIMpaClamp interface.
    • SMAMpaDisplacementRestraint : Creates a displacement restraint. It adheres to SMAIMpaDisplacementRestraint interface.
    • SMAMpaConnectorDisplacementRestraint : Creates a displacement restraint. It adheres to SMAIMpaConnectorDisplacementRestraint interface.
  • For Thermal Features :
    • SMAMpaFilmCondition : Creates a film condition. It adheres to SMAIMpaFilmCondition interface.
    • SMAMpaHeatFlux/b> : Creates a heat flux. It adheres to SMAIMpaHeatFlux interface.
    • SMAMpaTemperature : Creates a temperature. It adheres to SMAIMpaTemperature interface.
    • SMAMpaInitialTemperature : Creates an initial temperature. It adheres to SMAIMpaTemperature interface.
    • SMAMpaPrescribedTemperature : Creates a prescribed temperature. It adheres to SMAIMpaTemperature interface.
    • SMAMpaRadiationAmbient : Creates a radiation to ambient. It adheres to SMAIMpaRadiationAmbient interface.
    • SMAMpaRadiationCavityApprox : Creates a radiation cavity approx. It adheres to SMAIMpaRadiationCavityApprox interface.
  • For Output Features :
    • SMAMpaFieldOutput : Creates a field output. It adheres to SMAIMpaOuput interface.
    • SMAMpaHistoryOutput : Creates a history output. It adheres to SMAIMpaOuput interface.
  • For Load Set :
  • SMAMpaLoadSet : Creates a load set. It adheres to SMAIMpaLoadSet interface.
oFeature
[out] The created feature.
Returns:
S_OK if successful.

Parameters:
ispFeature

Returns:
S_OK if successful. E_FAIL if fail or if you try to delete a step.
