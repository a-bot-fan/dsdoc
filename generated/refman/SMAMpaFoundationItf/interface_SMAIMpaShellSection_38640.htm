enum IntegrationScheme {
  SimpsonIntegration,
  GaussIntegration
}
The method used to integrate the section.
Parameters:
SimpsonIntegration
Simpson's rule for integration.
GaussIntegration
Gauss's quadrature method.
o OffsetMethod 
enum OffsetMethod {
  None,
  SpecifiedDistanceOffset,
  ThicknessRatio,
  TopSurface,
  BottomSurface
}
The method used to offset the section.
Parameters:
None
No offset method is used for the section.
SpecifiedDistanceOffset
A specified distance offset method is used for the section.
ThicknessRatio
A thickness ratio offset method is used for the section.
TopSurface
A top surface offset method is used for the section.
BottomSurface
A bottom surface offset method is used for the section.
o PoissonMethod 
enum PoissonMethod {
  PoissonDefault,
  PoissonValue,
  PoissonMaterial,
  PoissonElastic
}
The method used to specify Poisson's ratio.
Parameters:
PoissonDefault
The default ratio is used for the section.
PoissonValue
A specified Poisson's ratio value is used for the section.
PoissonMaterial
A Poisson's ratio calculated from strain and material is used for the section
PoissonElastic
An elastic portion of material is used for the section.
