Methods


o GetReferencePoint
public virtual GetReferencePoint( ospPoint)
Retrieves the reference point. This method will only give a valid location if the ReferencePointInputMode is set to Specify. This method will return an empty reference point object when the location type is CenterOfMass.
Parameters:
ospPoint[out]
The point object for the reference point.
Returns:
S_OK if successful.
o GetReferencePointInputMode
public virtual GetReferencePointInputMode( oLocation)
Retrieves the reference point input mode.
Parameters:
oLocation[out]
The location type of the reference point.
Returns:
S_OK if successful.
o GetSupport
public virtual GetSupport( ospSupport)
Retrieves the support.
Parameters:
ospSupport
[out] The support of the rigid body constraint.
Returns:
S_OK if successful.
o SetReferencePointInputMode
public virtual SetReferencePointInputMode( iLocation)
Sets the reference point input mode. Specify means directly input by the user, either as point coords or as a point publication. CenterOfMass means that the analysis codes will compute the center of mass of the body and adjust the reference point coordinates.
Parameters:
iLocation[in]
The value of the location type.
Returns:
S_OK if successful.

Enumerated Types


o ReferencePointInputMode 
enum ReferencePointInputMode {
  Specify,
  CenterOfMass,
  Automatic
}
The mode used to locate the reference point.
Parameters:
Specify
The point location is input by the user.
CenterOfMass
The point is located at the center of mass of the body.
Automatic
Internal use only.
