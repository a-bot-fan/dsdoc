o GetMagnitude
public virtual GetMagnitude( oMassMagnitude)
Retrieves the mass magnitude.
Parameters:
oMassMagnitude
[out] The mass magnitude used for the nonstructural mass.
If the contribution type is TotalMass then the unit is : Quantity: MASS, units: Kg.
Else the unit depends of the feature's type :
  • MassPerVolume : Quantity: DENSITY, units: Kg_m3
  • MassPerArea : Quantity: SURFACICMASS, units: Kg_m2
  • MassPerLength : Quantity: LINEMASS, units: Kg_m
Returns:
S_OK if successful.

o ApplicationMethod 
enum ApplicationMethod {
  Mass,
  TotalMass
}
The method used to apply the mass.
Parameters:
Mass
The nonstructural mass is specified in the unit of mass per unit volume, area or length.
TotalMass
The nonstructural mass is specified in the unit of mass.
o GeometricType 
enum GeometricType {

}

o TotalMassDistributionMethod 
enum TotalMassDistributionMethod {
  MassProportional,
  VolumeProportional
}
The method used to distribute the mass when the application method is TotalMass.
Parameters:
MassProportionalMethod
The nonstructural mass is distributed in proportion to the element structural mass.
VolumeProportionalMethod
The nonstructural mass is distributed in proportion to the element volume.
