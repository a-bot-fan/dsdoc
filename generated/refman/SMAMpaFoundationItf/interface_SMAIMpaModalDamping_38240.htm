o GetTable
public virtual GetTable( ospTable)
Retrieves the table that determines the modal damping behavior.
The columns in the modal damping table will be:
For Fraction damping:
  1. Frequency values. Quantity: FREQUENCY, Units: Hz
  2. Damping Fraction values. Quantity: MODAL_DAMPING, Units: None

For Rayleigh damping:
  1. Frequency values. Quantity: FREQUENCY, Units: Hz
  2. Alpha values. Quantity: DAMPINGCOEFFICIENT, Units: _s
  3. Beta values. Quantity: TIME, Units: s
    For Structural damping:
    1. Frequency values. Quantity: FREQUENCY, Units: Hz
    2. Damping Factor values. Quantity: MODAL_DAMPING, Units: None
    Parameters:
    ospTable
    [out] The table that determines the modal damping behavior.
    Returns:
    S_OK if successful.

Enumerated Types


o DampingType 
enum DampingType {
  Fraction,
  Rayleigh,
  Structural
}
The damping type.
Parameters:
Fraction
Specify the fraction of critical damping for a particular frequency range.
Rayleigh
A general damping technique that provides a convenient way to damp lower (mass-dependent) and higher (stiffness-dependent) frequency range behavior.
Structural
Define damping that is proportional to the internal forces but opposite in direction to the velocity.
