o GetTable
public virtual GetTable( ospTable)
Retrieves the table that determines the elasticity behavior.
The columns in the connector elasticity table will be:
For Linear Elasticity:
  1. Stiffness values.
    • For translational DOF, Quantity: FORCE_OVER_DISPLACEMENT, Units: kg_s2
    • For rotational DOF, Quantity: FORCE_OVER_ROTATION, Units: m2kg_s2_rad
  2. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg

For NonLinear Elasticity:
  1. Stiffness values.
    • For translational DOF, Quantity: FORCE, Units: N
    • For rotational DOF, Quantity: MOMENT, Units: N_m
  2. Position values.
    • For translational DOF, Quantity: LENGTH, Units: m
    • For rotational DOF, Quantity: ANGLE, Units: deg
  3. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg
Parameters:
ospTable
[out] The table that determines the elasticity behavior.
Returns:
S_OK if successful.
