o DefinitionMode 
enum DefinitionMode {
  Global,
  Local,
  Specify
}
The mode used to define the FEA axis system.
Parameters:
Global
The used axis system is the global one.
Local
The used axis system is a local axis defined by the user.
Specify
The used axis system is defined by the application based on its internal data.
