o GetParameters
public virtual GetParameters( oParamValues)
Retrieves the parameters used to define the beam profile.
Parameters:
oParamValues
[out] The list of beam profile parameters. The possible values are:
  • Box : {Width, Height, Thickness1, Thickness2, Thickness3, Thickness4}
  • Circular : {Radius}
  • General : {Area, I11, I12, I22, J, GammaO, GammaW}
  • Hex : {Radius, Thickness}
  • I_Beam : {Length, Height, Width1, Width2, Thickness1, Thickness2, Thickness3}
  • L_Beam : {Width, Height, Thickness1, Thickness2}
  • Pipe : {Radius, Thickness}
  • Rectangular : {Base, Height}
  • Trapezoid : {Width1, Height1, Width2, Height2}
  • T_Beam : {Width, Height, Length, Thickness1, Thickness2}
Parameter Quantity Unit
Area AREA m2
I11, I12, I22, J and GammaO INERTIA m4
GammaW WARPING_COEFFICIENT m6
All other parameters LENGTH m
Returns:
S_OK if successful.
