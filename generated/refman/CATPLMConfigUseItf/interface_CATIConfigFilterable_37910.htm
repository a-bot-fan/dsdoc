interface CATIConfigFilterable

Interface to manage effectivity definition of a filterable object.

Methods


o GetEffectivity
public virtual GetEffectivity( out_pExpression,
in_effExpressionView= ConfigEffectivityView_Current )
Gets the Effectivity expression.
Parameters:
out_pExpression
[out] The expression in XML format as defined by the schema CfgEffectivityExpression.xsd
in_effExpressionView
[in] The effectivity view to get.
Returns:
An HRESULT value.
Legal values:
S_OK
The effectivity expression has been successfully retrieved.
E_FAIL
Unexpected failure or no Effectivity is defined on this Filterable.
o HasEffectivity
public virtual HasEffectivity()
Checks if there is an Effectivity defined.
Returns:
An HRESULT value.
Legal values:
S_OK
An effectivity is defined.
S_FALSE
No effectivity defined.
E_FAIL
Unexpected failure.
o IsFilteringCacheUpToDate
public virtual IsFilteringCacheUpToDate()
Checks if the cache of the Effectivity is up to date.
Returns:
An HRESULT value.
Legal values:
S_OK
The Effectivity is up-to-date.
S_FALSE
The Effectivity is not up-to-date.
E_FAIL
Unexpected failure or there is no Effectivity set on the Filterable.
