o LoadDictionaries
public static LoadDictionaries( in_configuredEntity,
io_sessionBag,
in_ifNotAlreadyInSessionOnly= TRUE,
io_pErrorReviewer= NULL)
Loads the dictionaries attached to a configured entity.
Parameters:
in_configuredEntity
[in] Configured entity that has dictionaries attached.
io_sessionBag
[in/out] Bag to put the loaded data in.
in_ifNotAlreadyInSessionOnly
[in] FALSE to force the loading even if the dictionaries are already in session. TRUE otherwise.
io_pErrorReviewer
[in/out] Error manager of load service.
Returns:
An HRESULT value.
Legal values:
S_OK
Operation succeeded and dictionaries loaded.
S_FALSE
Operation succeeded and no dictionary associated.
E_FAIL
Operation failed.
