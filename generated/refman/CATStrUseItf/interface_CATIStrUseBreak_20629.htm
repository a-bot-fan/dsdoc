o Break
public virtual Break( const iListOfSplittingObjects,
const iListOfOccurrencesOfSplittingObjects,
oListOfResults)
This method duplicates this element and applies new limits to all resulting elements (including this one) so that the result is the original object broken by the splitting elements.
Role: Break a Panel/Plate or a Profile.
Parameters:
iListOfSplittingObjects
[in] The list of features splitting this object.
iListOfOccurrencesOfSplittingObjects
[in] The list of product occurrences in which the splitting features are. In SFD, this list may remain empty. Otherwise, it has to be the same size as the first list.
oListOfResults
[out] The list of newly created Panels or Profiles.
Returns:
S_OK if everything ran ok. If the method fails you can obtain the diagnosis of the error by unstacking the error in the HRESULT.
