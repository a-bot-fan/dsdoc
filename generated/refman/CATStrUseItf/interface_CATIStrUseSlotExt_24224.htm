Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIStrUseSlotExt

Interface managing a slot.
Get a pointer to slot features and query to this interface. Role: This interface provides methods which helps in creation/edition of Slot/Slot With Composite Profile.
