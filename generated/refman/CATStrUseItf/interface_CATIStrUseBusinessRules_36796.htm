Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIStrUseBusinessRules

This interface, applicable for panel, plates, profiles, openings, flanges and endcuts in both SFD and SDD allows the caller to execute the business rules as defined in the data setup.

Method Index

Methods


o ExecuteAllBusinessRules
public virtual ExecuteAllBusinessRules()
Executes all business rules as defined in the data setup.
Supported business rules include:
Naming rule
Coloring rule
Pass-through point for opening rule
Index of layer for pass-through point rule
Returns:
S_OK if all defined rules were executed correctly, or if none is defined.
E_FAIL if one of the defined rules has encountered a failure.
This object is included in the file: CATIStrUseBusinessRules.h
If needed, your Imakefile.mk should include the module: CATStrUseItf
