CATPLMFastenerUseItf Enumeration CATFmaStkAssistantRowStatus

enum CATFmaStkAssistantRowStatus {
  FmaStkAssistantRow_NotTreated,
  FmaStkAssistantRow_SentForEdition,
  FmaStkAssistantRow_SentForCustoCommand,
  FmaStkAssistantRow_Edited,
  FmaStkAssistantRow_Deleted,
  FmaStkAssistantRow_Unknown
}
Status of the current fastener in the Stacking Assistant Command report.
Role: Each row of the Stacking Assistant command panel represents a Fastener+connection pair. This fastener can be at various stages during the execution of the command. These stages are as follows:
Values:
FmaStkAssistantRow_NotTreated
This Fastener+connection pair is not treated in the current scope of the command.
FmaStkAssistantRow_SentForEdition
This Fastener+connection pair is sent for Edition command (standard Live Edition command OR custo Edition command).
FmaStkAssistantRow_SentForCustoCommand
This Fastener+connection pair is sent for a custo command.
FmaStkAssistantRow_Edited
This Fastener+connection pair is edited in the current scope of the command.
FmaStkAssistantRow_Deleted
This Fastener is deleted from session.
FmaStkAssistantRow_Unknown
The status of this Fastener+connection pair cannot be determined.
