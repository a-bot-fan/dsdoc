module DS/i3DXCompassServices/i3DXCompassPubSub

The Pubsub is used to manage events between Compass and widgets.
It allows for a widget to subscribe, unsubscribe and publish events from the Compass.

Example 

require(['DS/i3DXCompassServices/i3DXCompassPubSub'], function(i3DXCompassPubSub) {

// ----- Listen to a Compass event -----
// launchAppCallback
    i3DXCompassPubSub.subscribe('launchAppCallback', function(data) {
        // Do something
        console.log('Response:', +JSON.stringify(data));
    });

// ----- Trigger an event on the Compass -----
// launchApp
    var params = {
         widgetId : 'Widget_Id',
         appId : 'X3DPLAY_AP',
         fileName : '-XXXFile',
         fileContent : 'content'
    }
    i3DXCompassPubSub.publish('launchApp', params);
});

Methods

<static> publish(topic, data)

Publish Compass event.

Parameters:
Name Type Description
topic

The event name.
The available events are:

Properties
Name Type Description
launchApp

Dispatch this event to launch a widget app/native app/web app
data

The available options. Each event has its own options:

  • launchApp event:
    Properties
    Name Type Argument Description
    appId String The application identifier to launch
    widgetId String <optional> The identifier of the widget that launched the application
    fileName String <optional> The identifier of the file name in the command line
    fileContent String <optional> The Content of the file that will be created

<static> subscribe(topic, callback)

Subscribe to Compass event.

Parameters:
Name Type Description
topic

The event name.
The available events are:

Properties
Name Type Description
launchAppCallback

Fired at App launching
callback

The callback function to execute when associated event (topic) is fired.

<static> unsubscribe(topic)

Unsubscribe Compass event.

Parameters:
Name Type Description
topic

The event name.
See the publish function for the available events.
This object is included in the framework i3DXCompassUI and the mweb module i3DXCompassServices
If needed, your Imakefile.mk should include the mweb module i3DXCompassServices and your IdentityCard should prerequire the framework i3DXCompassUI.
