module DS/PlatformAPI/PlatformAPI 

A module exposing some of the widget platform APIs.

They are usable both inside the platform you are requiring it and inside a same-origin iframe
(see https://developer.mozilla.org/en-US/docs/Web/Security/Same-origin_policy)

If used outside of a valid platform it will simply return `undefined` when called.

Methods

publish(topic, data)

Publish a message to the event queue. The message must have a topic. You can additionally add some data.

Parameters:
Name Type Argument Default Description
topic

The name of the topic to send message to.
data <optional>
 {}

An object representing the message body.
Example
            require('DS/PlatformAPI/PlatformAPI', function (API) {

                var self = {};
                self.onLoad = function () {
                    API.publish('MyWidgetName.launched');
                    API.publish('MyWidgetName.launched', { msg: 'loaded' });
                };

                widget.addEvent('onLoad', self.onLoad)

            });

subscribe(topic, callback) → {}

Subscribe to a topic from the event queue with the given callback. This will allow you to be notified of a message being sent on that topic and take action on it inside your callback. It is a good idea to have a unique topic since anyone can subscribe to your topic. Also it will avoid you receiving messages from others that might have chosen the same topic. For instance avoid topics like: widget or topic. Further information for this topic can be found inside the offcial documentation.

Parameters:
Name Type Description
topic

The name of the topic to subscribe to.
callback

A function to be called when a message for the specified topic is received. Will be passed the message body and the full message as arguments.
Returns:
  • The Subscription instance for easy unsubscribing afterward.
Type
Example
            require('DS/PlatformAPI/PlatformAPI', function (API) {

                var self = {};
                self.onLoad = function () {
                    API.subscribe('MyWidgetName.launched', function () {
                        widget.body.setHTML('Received message: launched');
                    });
                };

                widget.addEvent('onLoad', self.onLoad)

            });

unsubscribe(identifier)

Unsubscribe a previous subscription added with the subscribe() method. This actually removes the callback you previously placed on a topic.

Parameters:
Name Type Description
identifier |

The topic or the specific Subscription to unsubscribe.
Example
            require('DS/PlatformAPI/PlatformAPI', function (API) {

                var self = {};
                self.onRefresh = function () {
                    API.unsubscribe('MyWidgetName.launched');
                };

                widget.addEvent('onRefresh', self.onRefresh)

            });
This object is included in the framework WebappsUtils and the mweb module PlatformAPI
If needed, your Imakefile.mk should include the mweb module PlatformAPI and your IdentityCard should prerequire the framework WebappsUtils.
