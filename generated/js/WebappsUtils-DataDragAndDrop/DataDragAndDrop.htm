module DS/DataDragAndDrop/DataDragAndDrop 

A helper for playing with HTML5 drag and drop.

Will help you avoid common pitfalls through a simple API.

This will enable you to perform such behaviours:

* Saying that part of your application can receive a drag (be droppable) and be notified when it does
    * When a drag enter your dropzone
    * When a drag leaves your dropzone
    * When a drag hovers your dropzone
* Saying that part of your application can be draggable and be notified when it does
    * When a drag happens be notified when it stops
* Removing any drag events from an object

Source:

* http://www.quirksmode.org/blog/archives/2009/09/the_html5_drag.html
* https://www.inkling.com/blog/2013/10/html5s-drag-and-drop-problem/
* https://dev.opera.com/articles/drag-and-drop/

Methods

clean(element, type)

Remove any drag events added with on(), off(), draggable() or droppable() on the element passed.

Parameters:
Name Type Argument Description
element

A DOM element.
type <optional>

If type is 'drop' will remove only the drop event. If type is 'drag' will remove only the drag event. If not provided will remove both.

draggable(element, options)

Set the provided element as draggable, adding the necessary properties and binding the mandatory events. An element can only bet set draggable once. To change it, call clean first.

Parameters:
Name Type Description
element

A DOM element.
options

Additional options.

Properties
Name Type Description
data

The data you want to pass with this drag. Can be anything string like. eg. 'my secret data' or '{ "secret": "my secret data" }'
start

A function to call when drag starts for a draggable element. Will be passed the dragged element and the event.
stop

A function to call when drag end for a draggable element. Will be passed the dragged element and the event.

droppable(element, options)

Set the provided element as droppable, adding the necessary properties and binding the mandatory events. An element can only bet set droppable once. To change it, call clean first.

Parameters:
Name Type Description
element

A DOM element.
options

Additional options.

Properties
Name Type Description
enter

A function to call when the drag enter a droppable element. Will be passed the droppable element and the event.
leave

A function to call when the drag leave a droppable element. Will be passed the droppable element and the event.
over

A function to call when the drag hover a droppable element. Will be passed the droppable element and the event. This function is called each time the mouse moves. If it returns false, drop will not be called.
drop

A function to call when a drop occurs on a droppable element. Will be passed the dataTransfer, the element and the event.

off(dragElements, dropElements)

Remove the drag drop events attached to the provided elements.

Parameters:
Name Type Description
dragElements

An array of DOM nodes.
dropElements

An array of DOM nodes.

on(dragElements, dropElements, options)

Set both drag and drop items.

Parameters:
Name Type Description
dragElements

An array of DOM nodes.
dropElements

An array of DOM nodes.
options

Additional options.

Properties
Name Type Description
data

The data you want to pass with this drag.
start

A function to call when drag starts for a draggable element. Will be passed the dragged element and the event.
stop

A function to call when drag end for a draggable element. Will be passed the dragged element and the event.
enter

A function to call when the drag enter a droppable element. Will be passed the droppable element and the event.
leave

A function to call when the drag leave a droppable element. Will be passed the droppable element and the event.
over

A function to call when the drag hover a droppable element. Will be passed the droppable element and the event. This function is called each time the mouse move. If it returns false, drop will not be called.
drop

A function to call when a drop occurs on a droppable element. Will be passed the dataTransfer, the element and the event.
