module i18n

Example 

// for a NLS file in MyModule.mweb/assets/nls/test_en.json

{
    test: "ok",
    test_replace: "Hello {name} !"
}

// You can use the i18n plugin like below :

define("ds/MyModule/myComponent", [i18n!DS/MyModule/assets/nls/test], function (i18nMyModule) {
 // use the returned i18nMyModule as a JSON Object
 console.log(i18nMyModule.test) // show ok

 // use the returned i18nMyModule as an API
 console.log(i18nMyModule.get("test")) // Returns the value ok or the key name if they key doesn't exist
 console.log(i18nMyModule.replace(i18nMyModule.get("test_replace"), {name: "Ann O'nyme"})) // replace variable in a string
});

Methods

replace(message, values) → {}

Replaces the {key}s in a string by the corresponding one in values

Parameters:
Name Type Description
message

The message in which the values are replaced.
values

The values use as replacements.
Returns:
  • The message string with the replacements.
Type
Example
  i18n.replace('Hi {name} !', {name:'John'});
  // returns 'Hi John !'
This object is included in the framework WebappsUtils and the mweb module AmdLoader
If needed, your Imakefile.mk should include the mweb module AmdLoader and your IdentityCard should prerequire the framework WebappsUtils.
