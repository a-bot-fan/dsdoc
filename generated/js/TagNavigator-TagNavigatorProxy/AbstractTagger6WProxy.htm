class DS/TagNavigatorProxy/AbstractTagger6WProxy

Abstract class for TagNavigator proxy classes.

Methods

activate()

Activates the 6WTaggerProxy. It implies to show up the 6WTags of the proxy in the 6WTagger

addEvent(eventName, eventCallback, scope)

Sets callback functions, used for setting callbacks for filtering events

Parameters:
Name Type Argument Default Description
eventName

Name of the event you want to set a callback on. It can be

  • 'onFilterChange' if the proxy filteringMode is "FilteringOnServer"
  • 'onFilterSubjectsChange' if the proxy filteringMode is "WithFilteringServices"
eventCallback

Function called when the specified event is triggered. Its signature is depending on event:

  • filterChangeListener if the event is 'onFilterChange'
  • filterSubjectsChangeListener if the event is 'onFilterSubjectsChange'
scope <optional>
 undefined

Scope of the callback
Returns:

null

Example
     this.taggerProxy.addEvent('onFilterChange', eventCallback, this);
     this.eventCallback = function (filter) {
         // filter example "OnFilterChange":
         //{
         //    "allfilters": {
         //        "ds6w:when/ds6w:starts": [
         //            {
         //                "object": "2013",
         //                "type": "date"
         //            }
         //        ]
         //    }
         //}
         // filter example "OnFilterSubjectsChange":
         //{
         //   "filteredSubjectList": [
         //        'subject-1-uri',
         //        'subject-2-uri'
         //    ]
         //}
     };

deactivate()

Deactivates the 6WTaggerProxy. It implies to hide the 6WTags of the proxy in the 6WTagger

die()

Deletes the 6WTaggerProxy. First, it removes the 6WTags from the 6WTagger, and then deletes the connection from the 6WTagger. This proxy does not exist anymore for the 6WTagger. Usually this method is called when the widget is destroyed. You cannot use the proxy once the die method is called, all methods will fire an exception.

Returns:

null

focusOnSubjects(arrayOfSubjectUri)

Focus on subject(s), the 6WTagger will display only tags of the focused subject(s). To enable tag edition mode of the tagger: the widget has to be maximized, the proxy active, and subject(s) has to be focused.

Parameters:
Name Type Description
arrayOfSubjectUri

Array of subjects'URI to set the focus on.
Returns:

null

Example
 this.taggerProxy.focusOnSubjects(['subjectsURI-1', 'subjectsURI-2']);

getCurrentFilter() → {}

Gets the filter of the 6WTagger. The filter is the result of the end-user choice

Returns:

a filter6WTagger object

Type
Example
    var filter = taggerProxy.getCurrentFilter();
    //example of filter received
    filter = {
        "allfilters": {
              "ds6w:when/ds6w:starts": [
                  {
                      "disptext": "2013",
                      "object": "2013",
                      "type": "date"
                  }
              ]
          }
         "filteredSubjectList": ["subject-uri-1","subject-uri-2"]
      }

unfocus()

Unfocus on subject(s), the 6WTagger will display only tags of the focused subject(s).

Returns:

null
