module DS/TagNavigatorProxy/TagNavigatorProxy

Factory module to create an interface between a widget and the 6WTagger.

Example 

    var taggerProxy;
    require(['DS/TagNavigatorProxy/TagNavigatorProxy'], function (TagNavigatorProxy) {
        taggerProxy = TagNavigatorProxy.createProxy({...});
    });

Members

<readonly> filteringMode :

Filtering mode for proxy creation.

Type:
Properties:
Name Type Description
FilteringOnServer

Server side filtering
WithFilteringServices

Client side filtering done by 6WTagger. Means that data is completely available on client

Methods

createProxy(options) → {DS/TagNavigatorProxy/Tagger6WProxy|DS/TagNavigatorProxy/Tagger6WProxyWithFilteringServices}

Creates and returns a new Tagger6WProxy to communicate with the 6WTagger

Parameters:
Name Type Description
options
Properties
Name Type Argument Description
widgetId

Identifier of the widget, needed in order to retrieve the tab where the widget is instanciated
filteringMode filteringMode

Value is 'WithFilteringServices' for a client side filtering, or 'FilteringOnServer' for a server side filtering
tenant <optional>

Identifier of the platform from which subjects come from. Needs to be set if subjects come from the 3DEXPERIENCE PLATFORM
Returns:
Type
DS/TagNavigatorProxy/Tagger6WProxy | DS/TagNavigatorProxy/Tagger6WProxyWithFilteringServices
Example
     var options= {
         widgetId : widget.id,
         filteringMode : TagNavigatorProxy.FilteringOnServer,
         tenant: 'TENANT_ID'
     };
     var taggerProxy = TagNavigatorProxy.createProxy(options);
