SimMaterialOptions (Object)

Represents the collection of material options.
Example:
Given a SimMaterialDomain object you can retrieve a SimMaterialOptions collection as following: 
 Dim MyMaterialDomain As SimMaterialDomain
 ...
 Dim MyMaterialOptions As SimMaterialOptions
 Set MyMaterialOptions = MyMaterialDomain.Options

o Func Add( iType) As
Creates a new material option in a simulation domain.
Parameters:
iType
Type of the material option.
  1. SimIsotropicConductivity for conductivity.
  2. SimDensity for Density.
  3. SimElasticity for Elastic.
  4. SimExpansion for Expansion.
  5. SimGasketMembraneElastic for Gasket Membrane Elastic.
  6. SimGasketThicknessBehavior for Gasket Thickness Behavior.
  7. SimGasketTransverseShearElastic for Gasket Transverse Shear Elastic.
  8. SimHashinDamage for Hashin Damage.
  9. SimPlasticity for Plastic.
  10. SMAMatCastIronPlasticity for Cast Iron Plasticity.
  11. SMAMatDamping for Damping.
  12. SMAMatBulkModulus for Bulk Modulus.
  13. SMAMatSpecificHeat for Specific Heat.
  14. SMAMatInelasticHeatFraction for Inelastic Heat Fraction.
  15. SMAMatLatentHeat for Latent Heat.
  16. SMAMatVolumetricDrag for Volumetric Drag.
  17. SMAMatAcousticAbsorption for Acoustic Absorption.
Returns:
Created material option.
