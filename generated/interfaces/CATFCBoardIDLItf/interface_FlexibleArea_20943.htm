FlexibleArea (Object)


Property Index

AreaType
Gets and sets the Type of a constraint area
    The available values are :
  • OTHER_OUTLINE
  • ROUTE_OUTLINE
  • PLACE_OUTLINE
  • ROUTE_KEEPOUT
  • PLACE_KEEPOUT
  • PLACE_REGION
  • VIA_KEEPOUT
Heightmax
Gets and sets the Height max of the constraint area.
Heightmin
Gets and sets the Height min of the constraint area.
Identifier
Gets and sets the Identifier of the constraint area.
LAYER
Gets and sets the layer of the constraint area
    The available values are :
  • STANDART
  • BOTH
  • INNER
  • ALL
It depends on the the type of the constraints area according to the IDF format.
OWNER
Gets and sets the OWNER of the constraint area.
