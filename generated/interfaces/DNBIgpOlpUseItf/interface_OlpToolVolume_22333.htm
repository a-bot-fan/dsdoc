OlpToolVolume (Object)

A simplified tool volume.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
This object can be retrieved using

Property Index

Active
Get/Set whether the zone is active.
Index
Get the zone index.
IndexSet
Get whether an index is set for the zone.
NumShapes
Get the number of shapes.

Method Index

AddShape
Append a new shape.
GetShape
Get one of the shapes.
RemoveShape
Remove shape.

Properties


o Property Active() As
Get/Set whether the zone is active.
o Property Index() As
Get the zone index.
In some robot languages an index is used to identify the zone. If the Index property is not set (IndexIsSet returns FALSE), then getting the Index will fail.
o Property IndexSet() As
Get whether an index is set for the zone.
If not set, then getting the Index property will fail.
o Property NumShapes() As (Read Only)
Get the number of shapes.
