SimStaticStep (Object)

Represents the Static Step object.
Example:
Given a SimSteps object, you can create a SimStaticStep as following: 
 Dim MySteps As SimSteps
 ...
 Dim MyStaticStep As SimStaticStep
 Set MyStaticStep = MySteps.Add("SimStaticStep")
Given a SimSteps object, you can retrieve a SimStaticStep named "Static Step.1" as following: 
 Dim MySteps As SimSteps
 ...
 Dim MyStaticStep As SimStaticStep
 Set MyStaticStep = MySteps.Item("Static Step.1")
