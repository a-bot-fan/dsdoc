SimSteadyStateHeatTransferStep (Object)

Represents the Steady State Heat Transfer Step object.
Example:
Given a SimSteps object, you can create a SimSteadyStateHeatTransferStep as following: 
 Dim MySteps As SimSteps
 ...
 Dim MySteadyStateHeatTransferStep As SimSteadyStateHeatTransferStep
 Set MySteadyStateHeatTransferStep = MySteps.Add("SimSteadyStateHeatTransferStep")
Given a SimSteps object, you can retrieve a SimSteadyStateHeatTransferStep named "Steady State Heat Transfer Step.1" as following: 
 Dim MySteps As SimSteps
 ...
 Dim MySteadyStateHeatTransferStep As SimSteadyStateHeatTransferStep
 Set MySteadyStateHeatTransferStep = MySteps.Item("Steady State Heat Transfer Step.1")
