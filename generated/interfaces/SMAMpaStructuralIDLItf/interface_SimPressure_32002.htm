SimPressure (Object)

Represents the Pressure object.
Example:
Given a SimFeatures object, you can create a SimPressure as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyPressure As SimPressure
 Set MyPressure = MyFeatures.Add("SimPressure")
Given a SimFeatures object, you can retrieve a SimPressure named "Pressure.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyPressure As SimPressure
 Set MyPressure = MyFeatures.Item("Pressure.1")
