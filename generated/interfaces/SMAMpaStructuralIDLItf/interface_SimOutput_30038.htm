SimOutput (Object)

Represents the Output object.
Example:
Given a SimFeatures object, you can create a SimOutput as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyOutput As SimOutput
 Set MyOutput = MyFeatures.Add("SimOutput")
Given a SimFeatures object, you can retrieve a SimOutput named "Output.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyOutput As SimOutput
 Set MyOutput = MyFeatures.Item("Output.1")

Properties


o Property ExactTimeFlag() As
Returns or sets the flag that determines if the results are saved at the exact time.
TRUE: the results are saved at the exact time.
FALSE: the results are saved at the increment ending immediately after the time.
