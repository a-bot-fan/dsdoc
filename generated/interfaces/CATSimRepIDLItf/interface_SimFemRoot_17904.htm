SimFemRoot (Object)

Represents the root feature of the FEM Representation.
Role: It aggregates all the objects making up the FEM Representation.
Example: 
  This example shows how to retrieve the FEM Representation's root feature from its Representation Reference:
 
 Dim MyRepRef As VPMRepReference
 Set MyRepRef = ...
 Dim MyFEMRoot As SimFemRoot
 Set MyFEMRoot = MyRepRef.GetItem("SimFemRoot")
