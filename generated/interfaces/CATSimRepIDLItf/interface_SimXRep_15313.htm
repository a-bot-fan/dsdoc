SimXRep (Object)

Represents a XRep.

Property Index

FileName
Sets or retrieves the data file name associated to the XRep.

Method Index

AddRelation
Adds a relation to the XRep.
ExportFile
Exports the data of the XRep to a data file.
ImportFile
Imports a data file in the XRep.
RemoveRelation
Removes a relation from the XRep.

Properties


o Property FileName() As
Sets or retrieves the data file name associated to the XRep.

Methods


o Sub AddRelation( iPointedObject,
iRelationType)
Adds a relation to the XRep.
Parameters:
iPointedObject:
The link object that represent the pointed object.
iRelationType:
The kind of relation, depending on the linked objects.
o Sub ExportFile( iFolderPath)
Exports the data of the XRep to a data file.
Parameters:
iFolderPath:
The path of the folder in which the data file will be exported.
o Sub ImportFile( iFolderPath,
iKeepNavRep)
Imports a data file in the XRep.
Parameters:
iFolderPath:
The path of the folder from which the data file will be imported.
iKeepNavRep:
Tells whether the existing navrep (if any) will be kept or replaced by the default one.
o Sub RemoveRelation( iRelationPointedObject)
Removes a relation from the XRep.
Parameters:
iRelationPointedObject:
The link object that represent the pointed object, stored into the relation.
