SimResultantSensor (Object)

Represents the resultant sensor.
Role:After creating the resultant sensor, one needs to set the support using or .
Then set the frame selection to the sensor using the
and update the sensor using and the method.
Example:
 Given a SimResultsAnalysisCase object, you can create a resultant Sensor object as following.
 
 Dim oResultantSensor As SimResultantSensor
 Set oResultantSensor = ResultsAnalysisCase.CreateResultantSensor
 Dim oFrameSelection As SimFramesSelection
 Set oFrameSelection = ResultsAnalysisCase.CreateFrameSelector
 oFrameSelection.SymbolicFrameRange = SimAllFrames
 oResultantSensor.FrameSelector = oFrameSelection
 oResultantSensor.Update
