SimMathPoint (Object)

Represents the Math Point object.
Example:
Given a SimMathAxis object with an attribute Origin of type SimMathPoint, you can retrieve SimMathPoint object as following: 
 Dim MyMathAxis As SimMathAxis
 ...
 Dim MyMathPoint As SimMathPoint
 Set MyMathPoint = MyMathAxis.Origin
