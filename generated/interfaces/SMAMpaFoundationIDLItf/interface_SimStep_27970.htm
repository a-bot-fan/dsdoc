SimStep (Object)

Represents the Step object.
Example:
Given a SimSteps collection you can retrieve a SimStep object named as "Static Step.1" through the following code: 
 Dim MySteps As SimSteps
 ...
 Dim MyStep As CATBaseDispatch
 Set MyStep = MyAnalysisCases.Item("Static Step.1")
